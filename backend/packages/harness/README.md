# Harness Package

This directory contains the reusable Anvil harness package.

The canonical package root is:

- `backend/packages/harness/anvil/`

Runtime-owned subareas include:

- `agents/`
- `config/`
- `runtime/`
- `sandbox/`
- `tools/`
- `skills/`
- `memory/`
- `extensions/`
- `subagents/`
- `uploads/`
- `tracing/`

This layer is intentionally adapter-free.
